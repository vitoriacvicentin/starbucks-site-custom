# Starbucks site custom

This is a site inpired in starbucks, with react, next js and css.
See in : https://starbucks-site-custom.vercel.app/
## Installation

Installation Commands:

```bash
If your use npm:
npm install
```
```bash
If your use Yarn:
Yarn
```

## Usage in dev

```python
If your use npm:
npm run dev
```
```python
If your use yarn:
yarn run dev
```
